Elements of a Blog with CSS

Element One

Body Elements

The font is Open Sans Condensed, which I sourced from google fonts. I also added a navy background.

  • font-family: 'Open Sans Condensed', sans-serif;
  • background-color: navy;

    • Element Two

    Font Sizes

    The font sizes for the h1, p, and li elements are 25px, 20px and 15px respectively as a result of the CSS code font-size: 25px, 20px and 15px.

    Element Three

    Title Alignment & Weight

    The h1 elements are aligned and bolded as a result of the following CSS codes:

  • text-align: center;
  • margin-top: 10px;
  • font-weight: bold;

    • Element Four

    Blog Margins

    I added margins to the blog posts by assigning the following CSS codes to the p elements:

  • padding-left: 20px;
  • padding-right: 20px;

    • Element Five

    List Attributes

    Similarly, I added a color and margins to my lists with the following CSS codes:

  • color: blue;
  • padding-left: 25px;
  • padding-bottom: 20px;

    • Element Six

    Border, Background and Spacing between Blogs

    In order to create borders and spacing between the blogs, I used the following CSS codes under the section:

  • border: 2px solid gray;
  • background-color: white
  • margin: 20px;

    • Element Seven

    Creating a Unique Class for Subtitles

    Although both the subtitles and the information had p elements, I decided that I wanted the subtitles to have their own color. Consequently, I gave the subtitles their own class by adding class="subtitle" to the respective p elements, which allowed me to assign a unique color to the subtitles.

    Element Eight

    Alternating List Colors

    I wanted to implent nth of type in my code, so I used li:nth-of-type(even) in order to alternate the colors on my lists between blue and green.

    Element Nine

    First Letter Color Transposition

    Next, I used p.subtitle:hover:first-letter in order to transpose the colors of the first letter of the subtitle when you hover over them.

    Element Ten

    Inheritance

    I set the parent to orange and the child to inherit, which prompted the child to inherit orange.

    Element Eleven

    !Important

    I played with !Important, which resulted in a purple font.

    Element Twelve

    First-of-type

    Lastly, I used section:first-of-type to highlight the title of the page.
